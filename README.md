NAVIQ – Smart Navigation System
Project Overview

NAVIQ is a smart navigation system designed to help users efficiently navigate large environments such as campuses, hospitals, or corporate buildings. The system provides route guidance between locations and helps users quickly locate facilities.

To enhance usability, NAVIQ also integrates an AI-powered RAG Assistant (Retrieval-Augmented Generation) that allows users to ask questions about the campus, locations, or navigation-related information.
