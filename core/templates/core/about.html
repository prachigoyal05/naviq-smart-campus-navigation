Smart Campus Indoor Navigation System

Intelligent indoor navigation powered by QR positioning and AI assistance.

Overview

Designed for Seamless Campus Navigation

NAVIQ is an intelligent indoor navigation platform built to simplify campus movement for students and visitors. By combining graph-based pathfinding algorithms, QR-based positioning, and Retrieval-Augmented AI assistance, the system delivers precise directions and contextual information in real time.

  • Indoor Routing
  • Shortest Path Engine
  • QR Smart Positioning
  • AI Knowledge Assistant
  • Multi-Floor Support
Capabilities

Core Functional Modules

Smart Route Optimization

Implements Dijkstra’s algorithm to compute optimal indoor navigation paths using corridor connectivity graphs.

QR-Based Position Detection

Enables instant location mapping through strategically deployed QR nodes across buildings.

Interactive Map Interface

Dynamic map rendering with real-time path visualization and floor transitions.

RAG-Powered AI Assistant

Context-aware campus query resolution using a vector-based retrieval system connected to structured knowledge documents.

Architecture

Technical Implementation Stack

Database Layer

MySQL stores node coordinates, pathways, adjacency relationships, and structured campus metadata.

Backend Engine

Django manages API endpoints, routing logic, and shortest-path computation engine.

Frontend Interface

Responsive UI rendering interactive campus maps and navigation flows.

AI Intelligence Layer

Retrieval-Augmented Generation model integrated with campus documentation for contextual Q&A.
